# fizzer-socrates-it-2022

## Basic commands with Make

Run the `main` application with:
```sh
$ make run
```
and run `test` with:
```sh
$ make test
```

## Basic commands with Cabal

Run the `main` application with:
```sh
$ cabal run fizzbuzz
```
and run `test` with:
```sh
$ cabal run fizzbuzz-test
```
